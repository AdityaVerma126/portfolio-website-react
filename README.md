# Portfolio

Standalone React + Vite + Tailwind v4 portfolio. No Lovable required.

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Build
```bash
npm run build
npm run preview
```

## Deploy

### Vercel
1. Push this folder to a GitHub repo.
2. Go to vercel.com → New Project → Import the repo.
3. Vercel auto-detects Vite. Click Deploy.

### GitHub Pages / Netlify / Cloudflare Pages
- Build command: `npm run build`
- Output directory: `dist`

## Customize
Edit `src/components/Portfolio.tsx` — name, email, projects, links are at the top of the file.
